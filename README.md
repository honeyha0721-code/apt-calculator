# 원리금 계산기

대출금액, 기간, 금리와 상환방식을 입력하면 예상 월 납입금과 전체 상환 일정을 계산하는 반응형 웹페이지입니다.

## 기술 구성

- TypeScript
- Vite
- 반응형 HTML/CSS
- 서버 없이 브라우저에서 계산

## 지원하는 상환방식

- 원리금균등상환
- 원금균등상환
- 만기일시상환

모든 계산은 브라우저에서 처리되며 입력값을 별도 서버로 전송하지 않습니다.

## 로컬 실행

```bash
pnpm install
pnpm dev
```

배포 파일은 `pnpm build`를 실행하면 `dist` 폴더에 생성됩니다.

## GitHub Pages 공개

GitHub Actions로 `dist` 폴더를 배포하도록 구성할 예정입니다. Vite의 자원 경로는 GitHub Pages의 저장소 하위 주소에서도 동작하도록 상대 경로로 설정되어 있습니다.
