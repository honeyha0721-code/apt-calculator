# 원리금 계산기

대출금액, 기간, 금리와 상환방식을 입력하면 예상 월 납입금과 전체 상환 일정을 계산하는 정적 웹페이지입니다.

## 지원하는 상환방식

- 원리금균등상환
- 원금균등상환
- 만기일시상환

모든 계산은 브라우저에서 처리되며 입력값을 별도 서버로 전송하지 않습니다.

## GitHub Pages 공개

1. 이 폴더의 파일을 GitHub 저장소 기본 브랜치에 올립니다.
2. 저장소의 `Settings → Pages`로 이동합니다.
3. `Deploy from a branch`를 선택합니다.
4. 기본 브랜치와 `/ (root)` 폴더를 선택해 저장합니다.

별도 빌드 과정이나 서버가 필요하지 않습니다.
